# ArkInsight

AI-powered Application Security Assessment Platform.

## Vision

ArkInsight is an AI-assisted Application Security Assessment platform designed to help security professionals perform consistent, evidence-driven security assessments by combining structured security knowledge, AI reasoning, automated tooling, and professional reporting.


## Goals

- AI-assisted security assessments
- Knowledge-driven recommendations
- Findings management
- AI-generated reports
- Scanner integrations
- Multiple AI providers (OpenAI, Ollama, Manual Mode)
